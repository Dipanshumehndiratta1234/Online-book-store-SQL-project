# Online-book-store-SQL-project
Online Book Store – SQL Analysis Project
Project Description

This project contains SQL queries used to analyze an Online Book Store database.
The queries help extract useful insights such as best-selling books, customer
purchase behavior, revenue analysis, and inventory information.

The project is created for learning SQL data analysis and database querying.
Database Overview

Explain the tables used in the database.

The database includes the following tables:

- Books – stores book details like title, author, price, and stock
- Customers – contains customer information
- Orders – stores order details

Tables:
1. Books(book_id, title, author_id, price, stock)
2. Customers(customer_id, name, email)
3. Orders(order_id, customer_id, order_date)


Explain what analysis your queries perform.

This project includes SQL queries for:

- Total number of books in the store
- Total revenue generated
- Books with low stock
- Orders placed per month
- Author with most books sold
Tools Used
- SQL
- MySQL 
- Database management tools like MySQL Workbench
How to Use
1. Import the database schema.
2. Insert sample data into tables.
3. Run the SQL analysis queries provided in the project.
4. View results for business insights.
